🌪️ Y/W 介音矢量：多维扭曲与旋转

当基础元音前加入 [y] 或 [w]，动作不再是直线的，而是产生了加速度、扭曲或旋转包围。

  • ㅑ/ㅕ
    Y矢量 (瞬间加速/折叠)： 动作迅速弹开或剧烈扭曲收缩。
  • ㅘ/ㅝ
    W矢量 (全方位旋转/包裹)： ㅗ+ㅏ/ㅜ+ㅓ。代表力量的全面张开或巨大的漩涡内吸。
  • ㅐ/ㅔ
    混合端点 (方向+极点)： ㅏ+ㅣ/ㅓ+ㅣ。力量向某个方向延伸并最终聚焦于极小的一点。

⚙️ 辅音张力：破损与压力的量级

1挡 平音 (자르다, 가르다)

自然动作，平滑无阻碍。正常地分开、切开。

2挡 紧音 (짜다, 까다, 빨다, 빠르다)

气流死死憋住。极度压缩、坚硬、瞬间发力。挤、暴力剥壳、死死吸住、极速爆发。

3挡 送气音 (차다, 파다, 파르다)

气流喷涌而出。剧烈破坏、散开、带有风感。用力踢开、挖破地表、破裂开来。

【第五层】高阶合成系：多音节动作解析

뚫다
[ttulh-da] 贯穿
ㄸ(物理重压) + ㅜ(向深处) + ㄹ(持续钻进) + ㅎ(气流漏出)
施加极强的向下重压，让硬物向深处持续钻进滑动，直到彻底穿透让内部空气漏出。
껍질
[kkeop-jil] 壳/皮
껍(坚硬向内包裹形成表层) + 질(指尖捏住薄片剥落)
[ㄲ+ㅓ+ㅂ] 一个向内紧紧包裹的坚硬表层；[ㅈ+ㅣ+ㄹ] 可以用指尖捏住垂直分离并滑动撕扯下来的东西。
바람
[ba-ram] 风
바(广阔张开的平面) + 람(向外滚动的实体)
在无限敞开的平面上，有一股向外流动的势能，被整体封装封装(ㅁ)成了一个物理概念。
짓누르다
[jit-nu-reu-da] 蹂躏/重压
짓(向下的死死聚焦与摩擦) + 누르다(持续施加向下存在的压力)
[ㅈ+ㅣ+ㅅ]力量聚焦向下且带有摩擦破坏感；[ㄴ+ㅜ+ㄹ]力量深深沉入底部并持续滑动。极具压迫感的多音节组合。
얽매이다
[eolk-mae-i-da] 被束缚/缠绕
얽(无序滑动后卡死) + 매(向外延伸后封装) + 이(状态被动极点化)
[ㅇ+ㅓ+ㄺ]暗处的空虚之物在滑动(ㄹ)后被强行卡死(ㄱ)；[ㅁ+ㅐ]像容器一样包裹住延展物。极度痛苦的被动束缚态。
부스러기
[bu-seu-reo-gi] 碎屑
부(沉重膨胀) + 스(碎裂摩擦) + 러(延展) + 기(形成微小基点)
原本膨胀厚重的实体，被摩擦碾碎，碎片散开延展，最终变成了一个个微小的颗粒基点。
