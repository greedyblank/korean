韩语固有词音义推演全图谱

穷尽韩语 5 大发音器官（牙、舌、唇、齿、喉）、14 个基础辅音及其派生张力（送气、紧音）。
每一个辅音都是一个物理算子，结合元音的阴阳空间，生成了整个语言的词汇网。

【牙音 / 软腭音】 折角与边界

发音器官：舌根抵住软腭。 物理特征：硬度、转折点、边界形成、勾取。

ㄱ ㅋ ㄲ

ㄱ (G/K) 基础折角/边界

  • + ㅏ(阳) 가다 (去)：向外跨越边界。
  • + ㅏ(阳) 갈다 (磨/耕)：在硬物边界上持续(ㄹ)摩擦。
  • + ㅓ(阴) 거두다 (收)：从边界向内聚拢。
  • + ㅗ(阳) 고리 (钩子)：突出的硬角。
  • + ㅜ(阴) 구멍 (孔/洞)：向内凹陷的边界。
  • + ㅣ(点) 길 (路)：硬物延伸的线。

ㅋ (K') 送气：扩张/破裂的角

  • + ㅡ(阴) 크다 (大)：边界向外猛烈扩张到模糊。
  • + ㅐ(阳) 캐다 (挖)：强行挖开硬物表面。
  • + ㅗ(阳) 코 (鼻)：面部突出的、呼出气体的部位。

ㄲ (Kk) 紧音：极度坚硬/切断

  • + ㅏ(阳) 까다 (剥/砸)：暴力破坏极硬的壳。
  • + ㅏ(阳) 깎다 (削)：用锐利边缘瞬间切断物质。
  • + ㅡ(阴) 끊다 (断)：极度紧绷后瞬间断裂。
  • + ㅡ(阴) 끌다 (拉)：硬物在地上强行拖拽。

【舌音 / 齿龈音】 涌动、触碰与流转

发音器官：舌尖。 物理特征：N(绵延时间)、D(点接触/沉降)、L(顺滑滚动)。

ㄴ ㄷ ㄹ ㅌ ㄸ

ㄴ (N) 内在发生/缓慢流变

  • + ㅏ(阳) 나다 (产生)：内在的东西外显。
  • + ㅏ(阳) 날 (日/刃)：时光流逝/事物的延伸。
  • + ㅓ(阴) 너르다 (广阔)：向四面缓慢延伸。
  • + ㅡ(阴) 늙다 (老)：生命在阴性时间中的流逝。
  • + ㅜ(阴) 눕다 (躺)：身体向下缓缓沉降。

ㄷ (D) 实体点触/承载

  • + ㅏ(阳) 닿다 (触及)：实体点对点接触表面。
  • + ㅗ(阳) 돌 (石头)：沉重坚硬的实体承载物。
  • + ㅜ(阴) 두다 (放置)：将实体向阴性方位（下）放置。
  • + ㅡ(阴) 들다 (拿/进)：实体重量的承载/深入。

ㅌ / ㄸ 弹开/瞬间打击

  • + ㅏ(阳) 타다 (烧/乘)：瞬间弹开/触碰上升。
  • + ㅓ(阴) 털다 (抖落)：靠连续击打散去表面附着物。
  • + ㅏ(阳) 따다 (摘)：瞬间从接触点强行分离(紧音)。
  • + ㅡ(阴) 뜨다 (浮/睁)：对抗重力瞬间向上发力。

ㄹ (R/L) 滚动与流体（通常不作为固有词词头，多为后缀或辅音）

作为词尾或次级音节时，赋予词汇“绵延不断、圆润滚动、无摩擦滑过”的物理意义。
实例：구르다 (滚动 - 圆孔+流动)、흐르다 (流淌 - 虚空扩散+流动)、빠르다 (快 - 爆裂+顺滑无阻力)。

【唇音 / 双唇音】 容器、包裹与爆裂

发音器官：双唇。 物理特征：M(水/容纳/母亲)、B(张开平面/爆发)。

ㅁ ㅂ ㅍ ㅃ

ㅁ (M) 包容/聚集/水态

  • + ㅏ(阳) 마음 (心)：承载阳性情感的容器。
  • + ㅗ(阳) 모으다 (聚)：将分散物收拢。
  • + ㅗ(阳) 몸 (身体)：最高级的实体容器。
  • + ㅜ(阴) 물 (水)：最基础的聚集流体。
  • + ㅜ(阴) 묻다 (埋)：从所有方向包裹物体。
  • + ㅓ(阴) 먹다 (吃)：吸收入内部容器。

ㅂ (B) 外展/面/自然张开

  • + ㅏ(阳) 바다 (海)：极致平展广阔的平面。
  • + ㅏ(阳) 발 (脚)：向外伸出贴地的支撑面。
  • + ㅗ(阳) 보다 (看)：视线向外呈面状展开。
  • + ㅜ(阴) 불 (火)：向下扎根向上张开的能量。
  • + ㅣ(点) 비 (雨)：从面坠落至微小接触点。

ㅍ / ㅃ 破裂 / 极度紧绷

  • + ㅏ(阳) 파다 (挖)：[送气]强力破坏表层(破)。
  • + ㅣ(点) 피다 (绽放)：[送气]花朵冲破外皮散开。
  • + ㅏ(阳) 빨다 (吸吮)：[紧音]双唇极度紧绷压迫。
  • + ㅜ(阴) 뿌리다 (撒)：[紧音]力量压缩后瞬间向外喷射。

【齿音 / 摩擦音】 缝隙、生命与聚拢

发音器官：牙齿缝隙。 物理特征：S(分离出独立生命/细流)、J(精准聚拢/对焦)。

ㅅ ㅆ ㅈ ㅊ ㅉ

ㅅ (S) 分离生命/细碎穿透

  • + ㅏ(阳) 살 (肉/岁)：分离出的基础生命实体。
  • + ㅏ(阳) 사람 (人)：高阶生命实体(+M容器化)。
  • + ㅓ(阴) 서다 (站)：实体向下抵抗重力固定。
  • + ㅗ(阳) 솟다 (涌)：穿透缝隙向上喷出。
  • + ㅜ(阴) 숨 (呼吸)：生命实体内在的细微流动。

ㅈ (J) 聚焦/捏拢/归位

  • + ㅏ(阳) 잡다 (抓)：五指聚拢聚焦于一点。
  • + ㅏ(阳) 자다 (睡)：意识收拢，归位沉寂。
  • + ㅏ(阳) 작다 (小)：体积向中心收缩(阳性为实体小)。
  • + ㅗ(阳) 좁다 (窄)：空间向中心挤压聚拢。
  • + ㅣ(点) 지다 (输/落)：力量聚拢到极致后崩溃。

ㅆ / ㅊ / ㅉ 摩擦极值/冲天

  • + ㅡ(阴) 쓰다 (写)：[紧音]笔尖在表面持续且极高频的摩擦。
  • + ㅗ(阳) 쏟다 (倒)：[紧音]大量细碎液体瞬间挤出。
  • + ㅏ(阳) 차다 (踢/满)：[送气]聚拢的力量猛烈向外冲击。
  • + ㅏ(阳) 짜다 (挤/咸)：[紧音]空间极致压缩(挤)或味觉极高频刺激(咸)。

【喉音 / 声门音】 虚空、圆融与气态

发音器官：喉咙深处无阻碍。 物理特征：O(空、内核、原初)、H(气流、天空、消散)。

ㅇ ㅎ

ㅇ (Zero/Ng) 圆/核/无阻碍

  • + ㅏ(阳) 알 (蛋/核)：最原始的圆融闭合体。
  • + ㅏ(阳) 안 (内)：空心圆的内部。
  • + ㅏ(阳) 아기 (婴儿)：最原始初生的状态。
  • + ㅗ(阳) 오다 (来)：向中心（我）无阻碍地靠近。
  • + ㅜ(阴) 위 (上)：在垂直维度向虚空移动。
  • + ㅜ(阴) 울다 (哭)：情绪顺畅无阻地流泻。

ㅎ (H) 气/空/高/散发

  • + ㅏ(阳) 하늘 (天)：最高远、充斥气体的虚空。
  • + ㅐ(阳) 해 (太阳)：天空(ㅎ)散发的明亮能量。
  • + ㅏ(阳) 하다 (做)：动作在空间中散布(最抽象的泛化动词)。
  • + ㅡ(阴) 흐르다 (流)：气态般的物质不断向低处延展。
  • + ㅡ(阴) 흩다 (散)：物质被打成气态般的粉末散开。
  • + ㅣ(点) 희다 (白)：颜色虚化、散逸到极点(空无色)。

【复合元音 / 介音】 动态矢量与空间扭曲

物理特征：在基础元音前加入 /y/ 或 /w/ 的滑动。赋予词汇“旋转、加速、交织、多重化”的动态向量。

ㅑ ㅕ ㅘ ㅝ

Y 轴介音 (ㅑ, ㅕ, ㅛ, ㅠ) 交织 / 快速擦过 / 多重线索

加入 /y/ 滑音（对应 ㅣ 点），使原本直线的动作增加了“斜向、快速摩擦或两线交织”的物理感。

  • 얇다 (薄)：相比 '알다'(圆满)，加上 y 后变成被挤压拉扯的薄片。
  • 열다 (打开)：相比 '얼다'(冻结)，y 赋予了门扇沿轴心快速转开的动态。
  • 엮다 (编织)：线条(y)在空间中来回穿插交织。
  • 야위다 (消瘦)：生命体被侧向的力量快速削减。

W 轴介音 (ㅘ, ㅝ, ㅚ, ㅟ) 旋转 / 包裹 / 球形化

加入 /w/ 滑音（对应 ㅗ/ㅜ 圆唇），赋予动作或物体“旋转、立体环绕、从四面八方聚拢”的物理张力。

  • 꽉 (紧紧地)：相比 '깍'(硬切)，w 带来从四面八方360度绞紧的窒息感。
  • 감싸다 (包裹)：在 '싸다'(包) 的基础上，更强调立体缠绕。
  • 구르다 (滚动)：含有 ㅜ 的圆唇，赋予物体沿自身轴心转动的力学。
  • 휘다 (弯曲)：气流(ㅎ)加上旋转(w)，形成弧形扭曲的力。

【收音 / 韵尾】 终态决断与物理定格

物理特征：发音闭合时的状态。首辅音决定“动作如何开始”，收音决定“动作如何结束或物质的最终形态”。全系统归结为 7 种物理结局。

ㄱ ㄴ ㄷ ㄹ ㅁ ㅂ ㅇ

ㄱ (K) 截断 / 嵌合

气流瞬间切断，动作突然终止卡死。代表牢固固定、平面受阻。

  • 막다 (堵/挡)：动作进行到面(ㅁ)后被死死截断(ㄱ)。
  • 박다 (钉/镶)：张开(ㅂ)刺入后死死卡住(ㄱ)。
  • 벽 (墙)：展开的平面(ㅂ+ㅕ)作为绝对阻碍(ㄱ)。

ㄴ (N) 沉淀 / 绵延 / 留存

气流从鼻腔绵长流出。代表动作结束后余韵未消、状态稳定留存。

  • 산 (山)：拔地而起(ㅅ)后永远定格长存(ㄴ)的实体。
  • 안 (内部)：包容(ㅇ)并长久留存其间(ㄴ)。
  • 신 (神)：生命/气息(ㅅ+ㅣ)的永恒存在(ㄴ)。

ㄷ 系 (T) 触底 / 点对点封闭

舌尖抵住上颚。代表动作接触到底部、形成一个闭合的接触点。

  • 닫다 (关)：将两个实体点对点死死合拢(ㄷ)。
  • 곧 (马上)：时间线拉直(ㄱ)后瞬间触底(ㄷ)。
  • 끝 (尽头)：从极度紧绷(ㄲ)到彻底触底无路可走(ㅌ/t)。

ㄹ (L) 悬空 / 势能 / 流动结局

舌尖卷起不封死。代表悬而未决的势能、摇晃不定或流体的结局。

  • 물 (水)：聚集(ㅁ)后处于不断流变不定的状态(ㄹ)。
  • 매달다 (悬挂)：接触后下垂，处于摇晃势能状态(ㄹ)。
  • 길 (路)：硬向延伸(ㄱ+ㅣ)且通畅无阻一直连绵(ㄹ)。

ㅁ (M) 实体化 / 闭环完成

双唇死死闭合。将动词收束为具象实体，代表动作完成形成闭环。

  • 삶 (人生)：活着(살)这个动作的最终实体化总和(ㅁ)。
  • 꿈 (梦)：极度压缩的虚空(꾸)闭合成一个独立世界(ㅁ)。
  • 숨 (呼吸)：生命游丝(수)被容纳在胸腔容器中(ㅁ)。

ㅂ (P) 压迫面 / 瞬间捕获

双唇瞬间闭合压迫气流。代表受力面的形成、或者动作的瞬间抓取闭合。

  • 잡다 (抓)：五指聚拢(자)后瞬间施加面状压迫并捕获(ㅂ)。
  • 좁다 (窄)：空间向中心聚拢(조)形成紧迫的侧面压力(ㅂ)。
  • 입 (嘴)：人体上下的基准点(이)闭合产生的受力面(ㅂ)。

ㅇ (NG) 回声 / 球体化 / 膨胀与共振

气流在喉腔无阻碍共鸣。作为尾音，它将事物形态引向“圆球体”、或者动作产生“不断向外共振的余波”。

  • 공 (球)：硬壳边界(ㄱ)包围着膨胀的虚空(ㅇ)。
  • 방 (房间)：张开的四壁(ㅂ)围成的中空共鸣腔体(ㅇ)。
  • 둥글다 (圆)：像沉重的钟声(두)不断回荡(ㅇ)的形态。
  • 강 (江)：刻画大地的边界(가)并不断浩荡共鸣(ㅇ)的水体。
  • 동동 (咚咚)：拟声/拟态词，小球弹跳或漂浮时的共振。
