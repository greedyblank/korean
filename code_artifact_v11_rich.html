元音/介音系：天地人空间与多维扭曲

物理特征：基础元音构成天地人坐标系；介音(/y//w/)赋予动作旋转、加速、交织、多重化的动态向量。

📍 基础单极坐标系

ㅏ [a] 向外/明亮
例: 밝다 [박따] 明亮
ㅓ [eo] 向内/沉闷
例: 어둡다 [어둡따] 昏暗
ㅗ [o] 向上/凸起
例: 오르다 [오르다] 上升
ㅜ [u] 向下/深渊
例: 우울 [우울] 忧郁
ㅡ [eu] 水平/拉长
例: 늘 [늘] 总是
ㅣ [i] 垂直/焦点
例: 이 [이] 这个/存在

🌪️ Y/W 介音矢量：多维扭曲与旋转

当基础元音前加入 [y] 或 [w]，动作不再是直线的，而是产生了加速度、扭曲或旋转包围。加入 /y/ 滑音（对应 ㅣ 点），使原本直线的动作增加了"斜向、快速摩擦或两线交织"的物理感；加入 /w/ 滑音（对应 ㅗ/ㅜ 圆唇），赋予动作或物体"旋转、立体环绕、从四面八方聚拢"的物理张力。

  • ㅑ/ㅕ
    Y矢量 (瞬间加速/折叠)： 动作迅速弹开或剧烈扭曲收缩。例：얇다(薄-被挤压拉扯的薄片)、열다(打开-门扇沿轴心快速转开)、엮다(编织-线条来回穿插交织)、야위다(消瘦-侧向力量快速削减)、벼(水稻-平展面在风中弯曲摇曳)。
  • ㅘ/ㅝ
    W矢量 (全方旋转/包裹)： ㅗ+ㅏ/ㅜ+ㅓ。代表力量全面张开或漩涡内吸。例：꽉(紧紧地-从四面八方360度绞紧)、감싸다(包裹-立体缠绕)、구르다(滚动-沿自身轴心转动)、둥글다(圆-闭合环形)、와락(猛扑-全方位旋转包围网突然罩下)。
  • ㅐ/ㅔ
    混合端点 (方向+极点)： ㅏ+ㅣ/ㅓ+ㅣ。力量向某方向延伸并聚焦于一点。
