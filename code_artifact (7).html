【第一层】辅音宇宙：物理触感与动作地图

辅音（子音）是万物的“骨架”，决定了一个词的物理性质和动作方式。

👄 唇音家族 (双唇相碰)

核心意象：包容、平面、容器、向外发散

  • ㅁ (m) 〔水 / 聚集 / 内部容器〕

    물 (mul): 水(聚于容器) / 모으다: 收集 / 마음: 心(承载情感)

  • ㅂ (b/p) 〔平面 / 接触 / 底部〕

    바닥 (ba-dak): 地板(平整接触面) / 벽: 墙壁 / 발: 脚

  • ㅍ (p') 〔破裂 / 爆发 / 扩散〕

    피 (pi): 血(破皮而出) / 파도: 波浪 / 피다: 开花(向外爆开)

📐 舌根家族 (卡住喉咙)

核心意象：边缘、角度、外壳、断裂

  • ㄱ (g/k) 〔边缘 / 角度 / 外壳〕

    가 (ga): 边缘 / 껍질 (kkeop-jil): 硬壳 / 꺾다: 折断(产生直角)

  • ㅋ (k') 〔巨大的裂口 / 剧烈扩张〕

    크다 (keu-da): 大(超出边界) / 캐다: 挖掘(破坏硬壳)

👅 舌尖家族 (敲击上颚)

核心意象：大地、落脚、连接、平稳

  • ㄴ (n) 〔向下延伸 / 软性连接 / 自身〕

    나 (na): 我(指认存在) / 내리다: 降下(动作向下延伸)

  • ㄷ (d/t) 〔坚硬实体 / 稳固 / 封闭〕

    돌 (dol): 石头 / 닫다: 关门 / 땅 (ttang): 大地(最稳固基础)

  • ㄹ (r/l) 〔流动 / 循环 / 滚动〕

    흐르다 (heu-reu-da): 流淌 / 구르다: 滚动

🦷 牙齿/摩擦家族

核心意象：切割、尖锐、缝隙、精准

  • ㅅ (s) 〔渗透/缝隙〕 새(鸟/新) / 사이(间隙) / 스며들다(渗透)
  • ㅈ (j) 〔精准抓取〕 잡다(抓住) / 작다(捏在指尖的小)
  • ㅊ (ch') 〔强力刺穿〕 치다(击打) / 솟구치다(冲天而起)

🕳️ 喉音家族

核心意象：空洞、圆、空气、广阔

  • ㅇ (ng) 〔内部/浑圆〕 안(里面/包围虚空) / 원(圆圈)
  • ㅎ (h) 〔气流/发散〕 하늘(天空/充满气) / 한숨(叹息)

【第二层】元音坐标系：空间方向与体积

元音（母音）是万物的“血肉”，决定了词的方向感和体积大小。（无需死记阴阳，看“小短线”指哪即可）

➡️

ㅏ (a)

向右 / 向外

开放、明亮、轻快。
例：밝다 (明亮)

⬅️

ㅓ (eo)

向左 / 向内

收缩、暗沉、深沉。
例：어둡다 (昏暗)

⬆️

ㅗ (o)

向上 / 浮出

小巧、上升、表面。
例：오르다 (上升)

⬇️

ㅜ (u)

向下 / 下沉

沉重、下坠、深渊。
例：우울 (忧郁/下沉)

↕️

ㅣ (i)

垂直 / 存在

人站立样，锐利、指示。
例：이 (这个/此)

↔️

ㅡ (eu)

水平 / 底部

平坦大地，基底、拉长。
例：늘 (总是/时间拉长)

【第三层】物理张力调节阀 (韩语独有绝技)

改变一个词的“破坏力”或“坚硬度”，不需要换词，只需像挂挡一样调整发音的力度。

🚗 1 挡：平音 (松音)

松弛、自然流露的基础状态。

자르다 [ja-reu-da]

切开： 正常的切菜、剪纸等顺滑自然动作。

🚙 2 挡：紧音

憋气缩紧脖子，代表压缩、吃力、死死咬住。

짜다 [jja-da]

挤压： 比如用力挤出最后的牙膏、拧干抹布。

🏎️ 3 挡：送气音

向外喷气，代表破坏、爆破、猛烈。

차다 [cha-da]

用力踢开： 伴随着强气流的猛烈物理打击。

💡 小结：积木拼装法
当你看到 바다 (ba-da / 海)：
ㅂ (平面) + ㄷ (大地实体) = 平坦的实体
ㅏ (向外) + ㅏ (向外) = 无限敞开
组合起来：一望无际、平坦敞开的巨大实体 (海)。

【第四层】收音系统 (Batchim)：动作的物理终结

没有收音，动作就是发散未完成的。收音（韵尾）决定了一个物理动作如何“落地”或“锚定”，实现了组合数量的爆炸。

ㄱ (k) 闭锁 / 硬着陆

动作撞到硬墙死死挡住，形成固定实体。

  • 막다 [堵住]
    ㅁ(内)+ㅏ(向外)+ㄱ(闭锁) = 扩张时突遭卡死。
  • 벽 [墙壁]
    ㅂ(平面)+ㅕ(内部扩展)+ㄱ(闭锁)。

ㄴ (n) 沉淀 / 静态留存

动作轻轻落在表面，不再移动的持续状态。

  • 산 [山]
    ㅅ(尖锐)+ㅏ(向上/外)+ㄴ(沉淀) = 恒久沉淀的突起。
  • 눈 [雪/眼]
    ㄴ(向下)+ㅜ(深处)+ㄴ(附着) = 缓降附着物。

ㄹ (l/r) 悬浮 / 滚动势能

动作没有死死停住，保留了滑动流转的动能。

  • 돌 [石头]
    ㄷ(实体)+ㅗ(凸起)+ㄹ(滚动) = 坚硬可滚动的实体。
  • 길 [道路]
    ㄱ(线/边)+ㅣ(延伸)+ㄹ(延续滑动)。

ㅁ (m) 实体化 / 终极封装

将抽象动作彻底打包成一个具体的“包裹”(名词化)。

  • 잠 [睡眠]
    ㅈ(向下集中)+ㅏ(散开)+ㅁ(封装)。
  • 꿈 [梦]
    ㄲ(脱离现实)+ㅜ(深渊)+ㅁ(封装)。

ㅇ (ng) 空间共振 / 中空 / 悬停

动作终结在一个内部空旷的结构中，产生回音或膨胀感。

  • 방 [房间]
    ㅂ(墙面)+ㅏ(向外)+ㅇ(中空) = 推开留出的空洞。
  • 공 [球]
    ㄱ(硬壳)+ㅗ(凸起)+ㅇ(内部中空)。

【第五层】元音交替 (Ablaut)：微调旋钮

相同的辅音骨架，只需拨动元音的“阴阳旋钮”，就能衍生出无数个表示不同体积、重量、明暗的词汇。

阳性 (ㅏ, ㅗ)
轻、小、快、明亮、浅表
阴性 (ㅓ, ㅜ)
重、大、慢、暗沉、深渊
底层物理逻辑
졸졸 [jol-jol] 溪水潺潺流
줄줄 [jul-jul] 大水哗哗流
水流微调：【ㅈ(缝隙) + 元音 + ㄹ(滚动)】。ㅗ 感觉水流细小浅表；ㅜ 感觉水量沉重倾泻。
파랗다 [pa-rah-] 明亮的天蓝色
퍼렇다 [peo-reoh-] 沉重淤青的暗蓝
颜色微调：【ㅍ(爆发) + 元音 + ㄹ(延续) + ㅎ(发散)】。ㅏ 光线充足向外；ㅓ 光线暗沉重压。
발자국 [bal-ja-] 轻巧的浅脚印
벌자국 [beol-ja-] 成年人的深坑脚印
轨迹微调：【ㅂ(平面) + 元音 + ㄹ(延续)】。阳性 ㅏ 步态轻巧；阴性 ㅓ 沉重塌陷。
