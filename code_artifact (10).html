🌪️ Y/W 介音矢量：多维扭曲与旋转

当基础元音前加入 [y] 或 [w]，动作不再是直线的，而是产生了加速度、扭曲或旋转包围。

  • ㅑ/ㅕ
    Y矢量 (瞬间加速/折叠)： 动作迅速弹开或剧烈扭曲收缩。
  • ㅘ/ㅝ
    W矢量 (全方旋转/包裹)： ㅗ+ㅏ/ㅜ+ㅓ。代表力量全面张开或漩涡内吸。
  • ㅐ/ㅔ
    混合端点 (方向+极点)： ㅏ+ㅣ/ㅓ+ㅣ。力量向某方向延伸并聚焦于一点。

微调旋钮：张力齿轮与阴阳交替(Ablaut)

不换词根，只拨动参数：通过改变气流大小（紧音/送气）改变硬度，通过改变元音（阴/阳）改变体积大小。

⚙️ 辅音张力：破损与压力的量级

1挡 平音 (가르다)

自然动作，平滑无阻碍。正常地分开、切开。

2挡 紧音 (까다, 짜다)

声带憋紧。极度压缩、极致坚硬、瞬间发力。死死咬住、暴力剥壳、榨干。

3挡 送气音 (차다, 파다)

气流喷涌而出。剧烈破坏、散开、带有风感。用力踢开、挖破地表。

高阶合成系：多音节动词解剖

뚫다
[뚤타] ttulh-da
贯穿/打孔
ㄸ(物理重压) + ㅜ(向深处) + ㄹ(持续钻进) + ㅎ(气流漏出) 施加极强的向下重压，让硬物向深处持续钻进滑动，直到彻底穿透让内部空气漏出。
껍질
[껍찔] kkeop-jil

껍(包裹的硬壳) + 질(指尖剥落) [ㄲ+ㅓ+ㅂ] 一个向内紧紧包裹的极硬表层；[ㅈ+ㅣ+ㄹ] 可以用指尖捏住垂直分离并滑动撕下来的东西。
짓누르다
[진누르다] jit-nu-reu-da

짓(向下的死死聚焦与摩擦) + 누르다(持续向下存在) [ㅈ+ㅣ+ㅅ] 力量聚焦极点向下且带有尖锐破坏感；[ㄴ+ㅜ+ㄹ] 力量深深沉入底部并持续滑动。极具压迫感的多音节动作。
얽매이다
[엉매이다] eolk-mae-i-da

[부스러기] bu-seu-reo-gi
