核心边界声明：汉字词 (Sino-Korean) 排除原则

本底层架构图谱 仅适用于韩语“固有词” (고유어 / Goyu-eo)。韩语中有约 60% 的词汇是“汉字词”（如 산 [san] 山, 강 [gang] 江, 벽 [byeok] 壁），这些词的发音是对古代汉语的音译。它们的语义被锁定在表意的汉字字形上，绝对不适用于本系统的“物理-发音空间”矩阵推演。
本图谱所有示例均已进行严格筛查，确保全量为纯正的韩语本土固有词（如 뫼 [moe] 山, 가람 [ga-ram] 河）。

【牙音】 折角、硬度与边界

发音器官：舌根抵住软腭。 物理特征：硬度、转折点、边界形成、勾取。

ㄱ ㅋ ㄲ

ㄱ [g/k] 折角/边界

  • + ㅏ 가다 [ga-da] (去)：跨越当前边界向外。
  • + ㅗ 고리 [go-ri] (环/钩)：凸出的硬角。
  • + ㅜ 구멍 [gu-meong] (孔洞)：向内凹陷的边界。
  • + ㅡ 그다 [geu-da] (画线)：硬物在表面划出界限。

ㅋ [k'] 扩张/破裂的角 (送气)

  • + ㅡ 크다 [keu-da] (大)：边界向外猛烈扩张。
  • + ㅐ 캐다 [kae-da] (挖掘)：强行破开硬物表面。
  • + ㅗ 코 [ko] (鼻子)：面部突出的通气边界。

ㄲ [kk] 极度坚硬/切断 (紧音)

  • + ㅏ 까다 [kka-da] (剥/砸)：暴力破坏极硬的壳。
  • + ㅏ 깎다 [kkak-da] (削)：瞬间切断物质 (首尾极硬)。
  • + ㅡ 끊다 [kkeun-ta] (弄断)：极度紧绷后瞬间断开。

【舌音】 涌动、触碰与流转

发音器官：舌尖。 物理特征：N(绵延时间)、D(实体点触)、L(顺滑滚动)。

ㄴ ㄷ ㄹ ㅌ ㄸ

ㄴ [n] 内生涌动/缓慢流变

  • + ㅏ 나다 [na-da] (产生)：内在东西向外(a)显现。
  • + ㅏ 날 [nal] (日/刃)：时间或实体的延绵。
  • + ㅜ 눈 [nun] (眼/雪)：内在深处(u)留存(n)的感知晶体。
  • + ㅡ 늙다 [neulk-da] (老)：生命在阴性时间中流逝。

ㄷ [d] 实体点触/承载

  • + ㅏ 닿다 [da-ta] (触及)：实体点对点接触表面。
  • + ㅗ 돌 [dol] (石头)：沉重的实体承载物。
  • + ㅜ 두다 [du-da] (放置)：向阴性方位（下）放置。

ㅌ/ㄸ [t/tt] 弹开/瞬间打击

  • + ㅏ 타다 [ta-da] (烧/乘)：(送气) 瞬间弹开/升腾。
  • + ㅓ 털다 [teol-da] (抖落)：靠连续击打散去表面附着。
  • + ㅏ 따다 [tta-da] (摘)：(紧音) 从接触点强行分离。

【唇音】 容器、平面与爆裂

发音器官：双唇。 物理特征：M(水/容纳/母亲)、B(张开平面/爆发)。

ㅁ ㅂ ㅍ ㅃ

ㅁ [m] 包容/水/母体

  • + ㅏ 마음 [ma-eum] (心)：承载阳性情感的容器。
  • + ㅗ 몸 [mom] (身体)：最高级的实体容器。
  • + ㅜ 물 [mul] (水)：最基础的向下(u)流动的(l)聚集物。
  • + ㅓ 먹다 [meok-da] (吃)：吸收入内部容器并封底(k)。

ㅂ [b/p] 外展/面/自然张开

  • + ㅏ 바다 [ba-da] (海)：极致广阔的向外(a)平面(b)。
  • + ㅏ 발 [bal] (脚)：向外伸出贴地的支撑面。
  • + ㅗ 보다 [bo-da] (看)：视线向外呈面状展开。
  • + ㅜ 불 [bul] (火)：向下扎根但不断外卷(b)的流体(l)。

ㅍ/ㅃ [p'/pp] 破裂 / 极度紧绷

  • + ㅏ 파다 [pa-da] (挖)：(送气) 强力破坏表层使其破散。
  • + ㅣ 피다 [pi-da] (绽放)：(送气) 花朵冲破外皮散开。
  • + ㅃ 빨다 [ppal-da] (吸吮)：(紧音) 唇部极度紧绷压迫。
  • + ㅜ 뿌리다 [ppu-ri-da] (撒)：力量压缩后瞬间向外喷射。

【齿音】 缝隙、独立生命与聚焦

发音器官：牙齿缝隙。 物理特征：S(分离出的独立生命/细分)、J(精准聚拢/对焦)。

ㅅ ㅆ ㅈ ㅊ ㅉ

ㅅ [s] 分离出的生命/细微

  • + ㅏ 살 [sal] (肉/岁)：分离出的基础生命实体。
  • + ㅏ 사람 [sa-ram] (人)：最高阶生命实体(+M容器化)。
  • + ㅗ 손 [son] (手)：独立分离出(s)向上/外(o)的分支实体(n)。
  • + ㅓ 서다 [seo-da] (站立)：实体向下抵抗重力固定。

ㅈ [j] 聚焦/捏拢/归位

  • + ㅏ 잡다 [jap-da] (抓)：五指聚拢聚焦于一点(j)+合拢(p)。
  • + ㅏ 작다 [jak-da] (小)：体积向中心收缩。
  • + ㅗ 좁다 [jop-da] (窄)：空间向中心挤压聚拢。
  • + ㅣ 지다 [ji-da] (输/落)：力量聚拢到极致后崩溃。

ㅆ/ㅊ/ㅉ [ss/ch/jj] 摩擦极值/冲天

  • + ㅡ 쓰다 [sseu-da] (写/苦)：(紧音) 表面高频摩擦或刺激。
  • + ㅏ 차다 [cha-da] (踢/满)：(送气) 聚拢的力量猛烈外冲。
  • + ㅏ 짜다 [jja-da] (挤/咸)：(紧音) 空间极致压缩。

【复合元音】 动态矢量与空间扭曲

加入 /y/ (偏转/交织) 或 /w/ (旋转/包裹) 的滑动，赋予词汇多重向量。

ㅑ ㅕ ㅘ ㅝ

Y 轴介音 交织 / 侧擦 / 线索化

  • 얇다 [yalb-da] (薄)：相比 알다(圆)，y 带来挤压拉扯成薄片的向量。
  • 엮다 [yeok-da] (编织)：线条(y)在暗处(eo)来回穿插交织。
  • 열다 [yeol-da] (打开)：门扇沿轴心快速转开。

W 轴介音 旋转 / 环绕 / 立体包覆

  • 꽉 [kkwak] (紧紧地)：相比 깍(硬切)，w 带来四面八方360度的绞紧感。
  • 감싸다 [gam-ssa-da] (包裹)：在 싸다(包) 基础上强调立体缠绕。
  • 휘다 [hwi-da] (弯曲)：气流(h)加上旋转(w)，形成弧形扭曲的力。

얽 (eolk)

+

+
러 (reo)

+
기 (gi)

【收音】 终态决断与物理定格

首辅音决定“动作如何开始”，收音 (Batchim) 决定“动作如何结束”。

ㄱ ㄴ ㄷ ㄹ ㅁ ㅂ ㅇ

ㄱ [-k] 截断 / 嵌合

气流瞬间切断。代表牢固固定、平面受阻。

  • 막다 [mak-da] (堵)：动作进行到面(m)后被截断(k)。
  • 턱 [teok] (下巴)：向下突出(t)的实体硬边界(k)。

ㄴ [-n] 沉淀 / 绵延 / 留存

气流绵长流出。代表动作结束后余韵留存。

  • 안 [an] (内部)：包容(o)并长久留存其间(n)。
  • 신 [sin] (鞋)：分离脚部(s)并长久包裹留存(n)的实体。

ㄷ 系 [-t] 触底 / 封闭

舌尖抵住上颚。代表动作形成闭合接触点。

  • 닫다 [dat-da] (关)：实体点对点死死合拢(t)。
  • 끝 [kkeut] (尽头)：极度紧绷(kk)到触底无路可走(t)。

ㄹ [-l] 悬空 / 流动结局

悬而未决的势能、摇晃不定或流体的结局。

  • 물 [mul] (水)：聚集(m)后流变不定(l)。
  • 길 [gil] (路)：硬向延伸(g)且通畅无阻一直连绵(l)。

ㅁ [-m] 实体化 / 闭环完成

双唇死死闭合。代表动作完成形成实体闭环。

  • 삶 [salm] (人生)：“活(살)”动作的最终实体化(m)。
  • 꿈 [kkum] (梦)：紧缩虚空(꾸)闭合成独立世界(m)。

ㅂ [-p] 压迫面 / 瞬间捕获

代表受力面的形成、或者动作的瞬间抓取。

  • 잡다 [jap-da] (抓)：聚焦(자)后瞬间施加面状压迫(p)。
  • 입 [ip] (嘴)：基准点(이)闭合产生的受力面(p)。

ㅇ [-ng] 球体化 / 共振扩散

气流在喉腔无阻碍共鸣。形态引向“圆球体”或“共振的余波”。

  • 공 [gong] (球)：硬壳边界(g)包围着膨胀的虚空共振(ng)。
  • 방울 [bang-ul] (水滴/铃铛)：向外爆开的共振圆球(bang) + 向下流动的实体(ul)。
  • 둥글다 [dung-geul-da] (圆)：像沉重的钟声(dung)回荡的形态。
