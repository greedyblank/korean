系统边界声明：汉字词排除原则

本矩阵 仅适用于韩语“固有词” (고유어 / Native Korean Words)。韩语中约 60% 的词汇是“汉字词”（如 산 [san] 山，강 [gang] 江），它们是对古代汉语的音译，语义锁定在汉字字形上，绝对不适用于本系统的物理推演。
本图谱中所有示例（如 뫼 [moe] 山，가람 [ga-ram] 河）均已进行严格筛查，全量为纯正的本土词根。

牙音 (Velar) ㄱ ㅋ ㄲ

发音器官：舌根抵软腭。 物理特征：折角、硬物、边界划分、起步。

ㄱ [g/k]

基础/平缓

形成边界、硬物延伸、基础行进。

  • 가다 [ga-da]

    去/走：跨越当前边界向外(ㅏ)行进。

  • 고리 [go-ri]

    钩/环：向上/外(ㅗ)突出的硬质折角。

  • 길 [gil]

    路：硬质边界(ㄱ)平坦地延伸定格(ㄹ)。

  • 굳다 [gut-da]

    凝固：向内沉降(ㅜ)变硬(ㄱ)并封底(ㄷ)。

ㅋ [k']

送气/发散

边界向外猛烈扩张、破开硬物表面。

  • 크다 [keu-da]

    大：边界(ㄱ+风)向水平大地(ㅡ)无限扩张。

  • 코 [ko]

    鼻子：面部向上突出(ㅗ)且通气(ㅋ)的角。

  • 캐다 [kae-da]

    挖/采：带风地(ㅋ)破开表层(ㅐ)获取。

ㄲ [kk]

紧音/压缩

极致坚硬、瞬间的暴力切断或碰撞。

  • 까다 [kka-da]

    剥/砸：极度发力破坏表层硬壳(如剥栗子)。

  • 깎다 [kkak-da]

    削/剪：瞬间硬切(ㄲ)加上硬断尾音(ㄱ)。完美的一刀切。

  • 끝 [kkeut]

    尽头：硬度压缩(ㄲ)到水平极点(ㅡ)碰壁(ㅌ)。

  • 꺾다 [kkeok-da]

    折断：向内(ㅓ)瞬间施加极硬的力(ㄲ)截断(ㄱ)。

舌音 (Alveolar) ㄴ ㄷ ㄹ ㅌ ㄸ

发音器官：舌尖。 物理特征：绵延的流变(N)、实体点触(D/T)、顺滑流转(R/L)。

ㄴ [n]

内生/绵延
  • 나다 / 나 [na-da / na]

    产生 / 我：内在事物向外(ㅏ)显现的过程，或是自我本源。

  • 날 [nal]

    日子 / 太阳：内生显现(나)并在天空中连绵流转(ㄹ)。

  • 눈 [nun]

    眼睛 / 雪：内在深处(ㅜ)留存沉淀(ㄴ)的感知结晶。

ㄷ [d]

点触/承载
  • 닿다 [da-ta]

    触及：实体(ㄷ)在外部空间(ㅏ)产生物理接触。

  • 돌 [dol]

    石头：向上凸起(ㅗ)的沉重实体点(ㄷ)形态(ㄹ)。

  • 다리 [da-ri]

    腿 / 桥：支撑接触地面的点(다)并延伸(리)。

ㅌ / ㄸ [t / tt]

爆发/打击
  • 타다 [ta-da]

    燃烧/乘：(送气) 接触点瞬间弹开、气流升腾散开。

  • 터지다 [teo-ji-da]

    炸裂：向内(ㅓ)积压的力量猛烈爆开(ㅌ)。

  • 따다 [tta-da]

    摘取：(紧音) 从一个接触点上施加硬力，强行分离。

  • 뜨다 [tteu-da]

    升起/睁眼：在地平线(ㅡ)上瞬间克服重力崩脱(ㄸ)。

唇音 (Bilabial) ㅁ ㅂ ㅍ ㅃ

发音器官：双唇。 物理特征：包容的容器(M)、张开的平面(B)、喷发爆裂(P/PP)。

ㅁ [m]

容器/母体/水
  • 물 [mul]

    水：向下(ㅜ)流动的(ㄹ)聚合物(ㅁ)。

  • 몸 [mom]

    身体：向上凸起的最高级实体容器(ㅁ-ㅗ-ㅁ)。

  • 마음 [ma-eum]

    心：向外敞开(마)接纳后封闭(음)的情感容器。

  • 먹다 [meok-da]

    吃：吸收入内部(ㅓ)容器(ㅁ)并封底截断(ㄱ)。

ㅂ [b/p]

外展/平面
  • 바다 [ba-da]

    海：极致广阔的向外(ㅏ)延展的平面(ㅂ)。

  • 발 [bal]

    脚：向外伸出(바)贴地流转(ㄹ)的支撑面。

  • 보다 [bo-da]

    看：视线向上/前方(ㅗ)呈面状(ㅂ)展开接收。

  • 불 [bul]

    火：扎根向下(ㅜ)却不断外卷(ㅂ)的流体能量(ㄹ)。

ㅍ / ㅃ [p' / pp]

破裂/高压
  • 파다 [pa-da]

    挖：(送气) 强力破坏表层使其破散飞扬。

  • 피다 [pi-da]

    绽放：从微小极点(ㅣ)冲破外皮向外散开。

  • 빠르다 [ppa-reu-da]

    快：(紧音) 张开动作被瞬间压缩到极致。

  • 뼈 [ppyeo]

    骨头：深藏体内(ㅕ)极度坚硬紧绷(ㅃ)的核心面。

发音器官：牙齿缝隙摩擦。 物理特征：缝隙渗透、分离出的生命实体(S)、精准聚焦(J)。

ㅅ [s]

分离实体/生命
  • 살 / 사람 [sal / sa-ram]

    肉 / 人：从自然分离出的最高阶阳性生命实体。

  • 새 [sae]

    鸟 / 新：在空气间隙(ㅅ)中穿梭的扩展(ㅐ)实体。

  • 손 [son]

    手：独立分离出(ㅅ)并向上/外(ㅗ)延伸的肢体(ㄴ)。

  • 서다 [seo-da]

    站立：实体(ㅅ)向下(ㅓ)抵抗重力固定住。

ㅈ [j]

聚焦/捏拢
  • 잡다 [jap-da]

    抓：五指在表层(ㅏ)聚焦聚拢(ㅈ)并形成合围面(ㅂ)。

  • 작다 [jak-da]

    小：体积向中心聚焦收缩(ㅈ)并形成硬边界(ㄱ)。

  • 자다 [ja-da]

    睡：生命体神志收拢归位(ㅈ)。

ㅆ/ㅊ/ㅉ [ss/ch/jj]

摩擦极值
  • 쓰다 [sseu-da]

    写/苦：(紧音) 在水平面(ㅡ)上高频细微地摩擦刺入。

  • 차다 [cha-da]

    踢/满：(送气) 聚拢的力量瞬间带风向外(ㅏ)爆开。

  • 짜다 [jja-da]

    挤/咸：(紧音) 空间被压迫聚焦到极致挤出物质。

  • 찢다 [jjit-da]

    撕裂：极度紧缩(ㅉ)于一点(ㅣ)后分裂到底(ㅈ)。

收音 (Batchim) ㄱ ㄴ ㄷ ㄹ ㅁ ㅂ ㅇ

首辅音决定“动作如何开始”，收音决定“动作以何种物理形态结束”。

ㄱ [-k] 截断/嵌合

气流瞬间切断，代表牢固固定、受阻。

  • 막다 [mak-]：延展的面被截断(堵)
  • 턱 [teok]：向下突出的硬边界(下巴)

ㄴ [-n] 沉淀/留存

气流绵长，动作结束后余韵留存。

  • 안 [an]：长久留存其间(内部)
  • 신 [sin]：长久包裹脚部的实体(鞋)

ㄷ/ㅅ [-t] 触底/封闭

舌尖抵住上颚，动作形成闭合接触点。

  • 닫다 [dat-]：点对点死死合拢(关)
  • 끝 [kkeut]：极度紧绷到触底无路(尽头)

ㄹ [-l] 悬空/流动结局

悬而未决的势能、摇晃不定或流体。

  • 물 [mul]：流变不定(水)
  • 길 [gil]：通畅无阻一直连绵(路)

ㅁ [-m] 实体化闭环

双唇闭合，动作彻底完成并形成实体结果（动转名的核心机制）。

삶 [salm]：活(살)的实体化 → 人生 꿈 [kkum]：闭合虚空 → 梦 얼음 [eor-eum]：冻(얼)的实体化 → 冰

ㅇ [-ng] 球体化/共振扩散

气流在喉腔无阻碍共鸣，形态引向“圆球体”或“空旷的回声”。

공 [gong]：外壳包围共振虚空 → 球 방울 [bang-ul]：爆开的共振圆球 → 铃铛/水滴
