系统边界声明：汉字词排除原则

本矩阵 仅适用于韩语“固有词” (고유어 / Native Korean Words)。韩语中约 60% 的词汇是“汉字词”（如 산 [san] 山，강 [gang] 江），它们是对古代汉语的音译，语义锁定在汉字字形上，绝对不适用于本系统的物理推演。

polyline

牙音 (Velar) ㄱ ㅋ ㄲ

发音器官：舌根抵软腭。 物理特征：折角、边界划分、硬物延伸、阻碍。

ㄱ [g/k]

基础/平缓

形成边界、硬物延伸、基础行进。

  • 가다 [ga-da]

    去/走：跨越当前边界向外(ㅏ)行进。

  • 길 [gil]

    路：硬质边界(ㄱ)平坦地延伸定格(ㄹ)。

  • 가을 [ga-eul]

    秋天：事物边界变得清晰、冷峻的收割季节。

  • 걷다 [geot-da]

    卷起/步：将原本延展的平面向内(ㅓ)挂起/收拢。

  • 굳다 [gut-da]

    凝固：向内沉降(ㅜ)变硬(ㄱ)并封闭(ㄷ)。

ㅋ [k']

送气/发散

边界向外猛烈扩张、破开硬物表面。

  • 크다 [keu-da]

    大：边界(ㄱ+风)向水平大地(ㅡ)无限扩张。

  • 코 [ko]

    鼻子：面部向上突出(ㅗ)且通风(ㅋ)的硬角。

  • 캐다 [kae-da]

    挖/采：带风地(ㅋ)破开表层(ㅐ)获取内部物。

  • 칼 [kal]

    刀：带有破坏性气流(ㅋ)的向外(ㅏ)延伸之物(ㄹ)。

ㄲ [kk]

紧音/压缩

极致坚硬、瞬间的暴力切断、彻底压缩。

  • 까다 [kka-da]

    剥/砸：极度发力破坏表层硬壳(如剥栗子)。

  • 깎다 [kkak-da]

    削/剪：瞬间硬切(ㄲ)加上硬断尾音(ㄱ)。

  • 끝 [kkeut]

    尽头：硬度压缩(ㄲ)到水平极点(ㅡ)碰壁(ㅌ)。

  • 꿈 [kkum]

    梦：将沉重空间(ㅜ)极致闭合收缩(ㄲ+ㅁ)。

  • 꽉 [kkwak]

    紧紧地：硬力(ㄲ)伴随空间扭曲(ㅘ)后瞬间锁死(ㄱ)。

waves

舌音与流音 ㄴ ㄷ ㄹ ㅌ ㄸ

发音器官：舌尖抵住或弹击齿龈。 物理特征：内生流变(N)、实体点触(D/T)、顺滑流转(R)。

ㄴ [n]

内生/绵延
  • 나다 / 나 [na-da / na]

    产生 / 我：内在事物向外(ㅏ)显现，自我本源。

  • 날 [nal]

    日子 / 飞：内生事物在空间中连绵流转(ㄹ)。

  • 눈 [nun]

    眼 / 雪：内在深处(ㅜ)留存沉淀(ㄴ)的感知或结晶。

  • 늙다 [neulk-da]

    老：内在时间(ㄴ)沉重流过(ㅡㄹ)并停滞(ㄱ)。

  • 넣다 [neot-da]

    放入：引导事物向内/向下(ㅓ)潜入。

ㄷ [d]

点触/承载
  • 닿다 [da-ta]

    触及：实体(ㄷ)在外部空间(ㅏ)产生接触。

  • 돌 [dol]

    石头：向上凸起(ㅗ)的沉重实体点(ㄷ)。

  • 달 [dal]

    月亮：悬在外部高空(ㅏ)的点状悬浮物(ㄹ)。

  • 다리 [da-ri]

    腿/桥：支撑接触地面的点(다)并延伸(리)。

  • 닫다 [dat-da]

    关：两个点(ㄷ)在外部空间向中心强行合拢(ㄷ)。

ㄹ [r/l]

流转/不定

※ 韩语固有词极少以ㄹ开头，它主要作为“后缀/收音”，赋予流转、悬浮之意。

  • 흐르다 [heu-reu-da]

    流淌：气体/能量(ㅎ)化作液体顺滑地蔓延(르)。

  • 마르다 [ma-reu-da]

    干涸：实体(ㅁ)表面的水分流失掉(르)。

  • 물 / 길 / 달 [mul / gil / dal]

    作为收音：赋予水(不定型)、路(延绵)、月(悬浮态)的物理结局。

ㅌ / ㄸ [t / tt]

爆发/打击
  • 타다 [ta-da]

    燃烧/乘：接触点瞬间带风弹开、气流升腾。

  • 터지다 [teo-ji-da]

    炸裂：向内(ㅓ)积压的力猛烈爆开(ㅌ)。

  • 땀 [ttam]

    汗水：用力(ㄸ)挤出体内并在表面凝结(ㅁ)。

  • 땅 [ttang]

    大地：承受极强打击(ㄸ)依然广阔铺开(ㅇ)。

  • 따다 [tta-da]

    摘取：在接触点上施加硬力，强行分离。

  • 뜨다 [tteu-da]

    升起/睁眼：在水平线上瞬间克服重力崩脱。

唇音 (Bilabial) ㅁ ㅂ ㅍ ㅃ

发音器官：双唇闭合再放开。 物理特征：包容容器(M)、张开的平面(B)、喷发爆裂(P/PP)。

ㅁ [m]

容器/实体
  • 물 [mul]

    水：向下(ㅜ)流动的(ㄹ)聚合物(ㅁ)。

  • 몸 [mom]

    身体：向上凸起的最高级实体容器(ㅁ-ㅗ-ㅁ)。

  • 마음 [ma-eum]

    心：向外敞开接纳后封闭的情感容器。

  • 먹다 [meok-da]

    吃：吸收入内部(ㅓ)容器(ㅁ)并截断(ㄱ)。

  • 믿다 [mit-da]

    相信：极点信息(ㅣ)在内部容器(ㅁ)生根(ㄷ)。

ㅂ [b/p]

外展/平面
  • 바다 [ba-da]

    海：极致广阔的向外(ㅏ)延展的平面(ㅂ)。

  • 발 [bal]

    脚：向外伸出贴地流转的支撑面。

  • 비 [bi]

    雨：在无数小极点(ㅣ)上铺开洒下(ㅂ)。

  • 별 [byeol]

    星：在暗夜(ㅕ)空中张开闪烁的发光体。

  • 불 [bul]

    火：扎根向下却不断外卷的流体能量。

ㅍ / ㅃ [p' / pp]

破裂/高压
  • 파다 [pa-da]

    挖：(送气) 强力破坏表层使其破散飞扬。

  • 피다 [pi-da]

    绽放：从微小极点(ㅣ)冲破外皮散开。

  • 빠르다 [ppa-reu-da]

    快：张开动作被瞬间压缩到极致。

  • 뼈 / 뿔 [ppyeo / ppul]

    骨/角：深藏体内或凸出体外，极度紧绷压缩的材质。

  • 뿜다 [ppum-da]

    喷射：极高压(ㅃ)液体向外释放并形成实体(ㅁ)。

air

齿音 (Sibilant) ㅅ ㅆ ㅈ ㅊ ㅉ

发音器官：齿缝摩擦。 物理特征：渗透、生命实体析出(S)、聚焦收拢(J)。

ㅅ [s]

分离/生命体
  • 살 / 사람 [sal / sa-ram]

    肉 / 人：从自然分离出的最高阶生命实体。

  • 새 [sae]

    鸟 / 新：在空气缝隙中穿梭的扩展实体。

  • 손 [son]

    手：独立分离并向上延伸的肢体。

  • 소리 [so-ri]

    声音：透出身体(ㅅ)在空气中蔓延(리)的波。

  • 숨 [sum]

    呼吸：向下沉入(ㅜ)生命体并闭合(ㅁ)的气流。

ㅈ [j]

聚焦/捏拢
  • 잡다 [jap-da]

    抓：五指在表层(ㅏ)聚拢(ㅈ)形成合围面(ㅂ)。

  • 작다 [jak-da]

    小：体积向中心聚焦收缩(ㅈ)碰到边界(ㄱ)。

  • 자다 [ja-da]

    睡：生命体神志收拢归位。

  • 젖 [jeot]

    乳汁：在暗处内部(ㅓ)极度浓缩聚焦的营养体。

  • 지다 [ji-da]

    落/背负：汇聚成点(ㅣ)后沉降。

ㅆ/ㅊ/ㅉ [ss/ch/jj]

摩擦极值/破裂
  • 쓰다 [sseu-da]

    写/苦：在平面上高频细微地摩擦刺入。

  • 씻다 [ssit-da]

    洗：紧密高频地(ㅆ)搓揉表面微小处(ㅣ)。

  • 차다 / 찾다 [cha-da / chat-da]

    踢/找：聚拢的力量带风向外爆开去索取。

  • 짜다 [jja-da]

    挤/咸：空间被压迫聚焦到极致挤出物质。

  • 찢다 [jjit-da]

    撕裂：极度紧缩于一点(ㅣ)后分裂到底。

lens_blur

喉音 (Glottal) ㅇ ㅎ

发音器官：喉腔中空。 物理特征：完全无阻碍的虚空、圆融状态(NG)与能量耗散、气流消退(H)。

ㅇ [ng / null]

虚空/圆融/内生

作为首音它不发音，代表事物处于“原始、不受外力干扰”的自然空心状态。

  • 알 [al]

    蛋 / 颗粒：最原始的虚空(ㅇ)向外显现(ㅏ)并形成独立流转体(ㄹ)。

  • 안 [an]

    内侧：虚空的外壳内部留存(ㄴ)的空间。

  • 오다 [o-da]

    来：不受阻力地从高空/远处(ㅗ)降临靠近。

  • 얼 [eol]

    精神/灵魂：深藏在暗处(ㅓ)的无形虚空悬浮物(ㄹ)。

ㅎ [h]

耗散/气流/光

气流呼出消散。代表能量的释放、光芒、气体的蔓延或实体的虚化。

  • 해 / 하얗다 [hae / ha-ya-ta]

    太阳 / 纯白：无尽释放光芒与气流(ㅎ)的极致外展(ㅏ)。

  • 하다 [ha-da]

    做 / 行为：向外(ㅏ)释放能量(ㅎ)的最广义动作。

  • 흐르다 [heu-reu-da]

    流淌：能量化作无形状态(ㅡ)不断蔓延(ㄹ)。

  • 혀 / 허리 [hyeo / heo-ri]

    舌头 / 腰：身体中具有柔软、灵活、如同气流般扭动特性的中空/虚位器官。

cyclone

复合矢量层 (Glides) Y系 (ㅑㅕㅛㅠ) / W系 (ㅘㅝㅚ)

介音并非单纯的元音拼接，它们在物理上等同于为辅音施加了“扭曲、摩擦(Y)”或“旋转、包围(W)”的受力矢量。

Y 矢量 [ya/yeo/yo/yu]

扭曲/编织/交错

口腔发生急速滑动，赋予动作一种**非直线、弯曲、交错纠缠**的质感。

  • 엮다 [yeok-da]

    编织/纠缠：虚空事物(ㅇ)发生严重交错扭曲(ㅕ)并打结(ㄲ)。

  • 야위다 [ya-wi-da]

    消瘦：原本饱满的实体发生了扭曲干瘪(ㅑ)。

  • 벼 [byeo]

    水稻：平展面(ㅂ)在风中弯曲摇曳(ㅕ)的状态。

W 矢量 [wa/wo/wi]

旋转/席卷/涡流

嘴唇从圆唇突变为展唇，赋予动作**巨大的旋转包裹感、全方位的席卷**。

  • 돌다 / 둥글다 [dol-da / dung-geul-da]

    转圈 / 圆：实体(ㄷ)伴随圆唇(ㅗ/ㅜ)带来的环形轨迹。

  • 와락 [wa-rak]

    猛扑/席卷：完全空旷的(ㅇ)全方位旋转包围网(ㅘ)突然罩下。

  • 위 [wi]

    上面：在基准点(ㅣ)周围形成全包裹态(ㅜ)的高层维度。

收音定格 (Batchim) ㄱ ㄴ ㄷ ㄹ ㅁ ㅂ ㅇ

首辅音决定“动作如何开始”，收音决定“动作以何种物理形态结束”。

ㄱ [-k] 截断/嵌合

气流瞬间切断，代表牢固固定、受阻。

  • 막다 [mak-]：延展的面被截断(堵)
  • 턱 [teok]：向下突出的硬边界(下巴)

ㄴ [-n] 沉淀/留存

气流绵长，动作结束后余韵留存。

  • 안 [an]：长久留存其间(内部)
  • 신 [sin]：长久包裹脚部的实体(鞋)

ㄷ/ㅅ [-t] 触底/封闭

舌尖抵住上颚，动作形成闭合接触点。

  • 닫다 [dat-]：点对点死死合拢(关)
  • 끝 [kkeut]：极度紧绷到触底无路(尽头)

ㄹ [-l] 悬空/流动结局

悬而未决的势能、摇晃不定或流体。

  • 물 [mul]：流变不定(水)
  • 길 [gil]：通畅无阻一直连绵(路)

ㅁ [-m] 实体化闭环

双唇闭合，动作彻底完成并形成实体结果（动转名的核心机制）。

삶 [salm]：活(살)的实体化 → 人生 꿈 [kkum]：闭合空间 → 梦 얼음 [eor-eum]：冻(얼)的实体化 → 冰

ㅇ [-ng] 球体化/共振扩散

气流在喉腔无阻碍共鸣，形态引向“圆球体”或“空旷的回声”。

공 [gong]：外壳包围共振虚空 → 球 방울 [bang-ul]：爆开的共振圆球 → 铃铛/水滴
